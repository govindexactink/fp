Bulk Location Upload Help

Enter a list of locations (one per line) to add them to your profile.

  • Enter specific area/neighborhood names (e.g., "Downtown", "Capitol Hill", "Mission District")
  • Or enter a city name (e.g., "Los Angeles", "Seattle") to add all neighborhoods in that city
  • Or enter a state name (e.g., "California", "Washington") to add all locations in that state
  • The system will match against known locations and add them with their zipcodes
  • Use the "Restricted Locations" field to specify locations that should NOT be added
  • Already-existing locations will appear in the list with an option to EXCLUDE them from your profile
  • After processing, use checkboxes to Include (add new) or Exclude (remove existing) locations
  • Use "Include All"/"Exclude All" buttons for quick selection
